magneticalc.Metric.Metric class documentation

Part of magneticalc.Metric (View In Hierarchy)

Provides different metrics, mapping some field vector properties to some color and alpha range. For a list of color maps, see: https://matplotlib.org/3.1.1/gallery/color/colormap_reference.html
