def set_wire(self, points=None, stretch=None, rotational_symmetry=None, close_loop=None, slicer_limit=None, dc=None, recalculate=(True), readjust_sampling_volume=(True), invalidate_self=(True)):

Sets the wire. This will overwrite the currently set wire in the model. Any parameter may be left set to None in order to load its default value.

Note: Currently, only `stretch` and `rotational symmetry` controls are automatically updated by this function.

ParameterspointsPoints (List of 3D points)
stretchXYZ stretch transform factors (3D point)
rotational_symmetryDictionary for rotational symmetry transform
close_loopEnable to transform the wire into a closed loop (append first point)
slicer_limitSlicer limit
dcDC value
recalculateEnable to trigger final re-calculation
readjust_sampling_volumeEnable to readjust sampling volume
invalidate_selfEnable to invalidate the old wire before setting a new one
