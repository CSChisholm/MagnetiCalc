magneticalc.Model.Model class documentation

Part of magneticalc.Model (View In Hierarchy)

Model class. The model maintains a strict hierarchy of dependencies: parameters > metric > field > sampling volume > wire When a lower module's data changed, all higher modules are invalidated (i.e. have their calculation results reset).
